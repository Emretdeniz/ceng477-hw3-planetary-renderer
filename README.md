# Planetary Body Renderer (CENG 477 HW3)

OpenGL planetary body visualizer featuring hierarchical animation (Earth–Moon–Moon's Moon), textures, and shader effects.

## Assignment PDF
- **Full spec (PDF):** [CENG_477_HW3_2025_Fall.pdf](CENG_477_HW3_2025_Fall.pdf)

## Build & Run
```bash
cmake -B build .
cmake --build build
./working_dir/PlanetRenderer
```

## Notes
- Requires a C++ toolchain + OpenGL-capable GPU/driver.
- Controls and extra details are described in the PDF.
